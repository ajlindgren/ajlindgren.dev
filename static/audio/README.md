# Audio samples

Drop 4–6 short (~300ms) dice-hit MP3s here, named:

- `clatter_01.mp3`
- `clatter_02.mp3`
- `clatter_03.mp3`
- `clatter_04.mp3`
- `clatter_05.mp3`
- `clatter_06.mp3`

Recommended source: [Freesound.org](https://freesound.org/) → search "dice hit" or "wood knock". Filter to CC0.

Trim to ~300ms, normalize to -3 dBFS, export as MP3.

If files are missing, `playRandomHit()` silently no-ops; the die still tumbles but is mute.

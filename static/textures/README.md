# Textures

Drop 2K hardwood PBR textures here:

- `hardwood_albedo.jpg`
- `hardwood_normal.jpg`
- `hardwood_roughness.jpg`

Recommended source: [ambientCG](https://ambientcg.com/) → search "wood floor" → download 2K JPG. Rename the files to the names above.

If files are missing, the floor falls back to a flat brown color; the site still works.
